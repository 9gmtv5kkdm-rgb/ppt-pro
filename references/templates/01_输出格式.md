## 📤 输出格式

### 标准输出格式（三部分）

标准输出由「标题 / 要点 / 图示建议」三部分组成。下面是一页 PPT 大纲的 Markdown 结构样例：

```markdown
# 幻灯片标题：2024 年度产品复盘

## 核心要点
- 营收同比增长 32%，超出目标 8 个百分点
- 新增付费用户 12.6 万，主要来自华东与华南
- 老客复购率提升至 47%，留存显著改善

## 支撑数据
- Q1–Q4 营收：0.8 / 1.1 / 1.4 / 1.9 亿元（逐季上升）
- 用户结构：新客 53% · 老客 47%（环形占比）

## 图示建议
- 左侧：营收季度趋势 → 折线图（强调 Q4 拐点）
- 右侧：用户构成 → 环形图（突出老客占比）
- 底部：结论占位条，承载本页关键 takeaway
```
