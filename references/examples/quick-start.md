# 5 分钟快速上手：把一句话需求做成 PPT（walkthrough）

> 本文件是外壳 [`SKILL.md`](../../SKILL.md) 中「5 分钟快速上手」的展开版。跟着走，5 分钟拿到可制作的结构化大纲。

## 场景

用户说：**「帮我做个 10 页的安全生产培训 PPT，给车间新员工看。」**

## 步骤

### 1. 锁定场景与受众（S3 教学课件 + S6 培训）

- 场景：培训材料
- 受众：车间新员工（非专业背景）
- 目标：让员工知道隐患在哪、遇到危险怎么办

### 2. 搭骨架（金字塔逻辑 · 五层结构法）

封面 → 为什么安全重要 → 六类常见隐患 → 遇到危险怎么办 → 你的责任 → 结尾口号

### 3. 生成大纲（每页一句话观点型标题）

1. 封面：安全生产，从认识风险开始
2. 开场：去年车间 3 起小事故，本可避免
3. 为什么：安全不是约束，是保护自己和同事
4. 过渡：先认识六类常见隐患
5. 六类常见隐患：机械 / 用电 / 高空 / 化学品 / 消防 / 人为
6. 遇到危险怎么办：停—报—撤三步法
7. 你的责任：上岗前 5 个必查
8. 误区：这 3 种「偷懒」最致命
9. 发现隐患不说最危险：人人都是安全员
10. 结尾：安全回家，是对家人最好的交代

### 4. 给每页内容提示 + 设计建议

- 第 5 页：用「六宫格对比图」并列六类隐患，每格 1 图标 + 1 句话。
- 配色：警示橙 + 深灰底，忌花哨；字体：标题 32pt 加粗，正文 20pt。
- 动画：仅「出现」即可，不做炫技转场。

### 5. 落地

把大纲交给 PowerPoint / 前端工具制作，需要改哪页直接说。

完整范例（含每页详细内容提示与设计建议）看 [`../cases/01_完整示例.md`](../cases/01_完整示例.md)。
